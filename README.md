# Mapping_Earthquakes

# Overview
The purpose of this assignment was to create a web page that demonstrate a custom map for all the earthquakes in the world as well as a map of the tectonic plates.
#### Tools Used
JavaScript, Leaflet, Mapbox, HTML.
